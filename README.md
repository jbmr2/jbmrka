# JBMR Raid Arena

Professional Kabaddi Tournament Management and Live Scoring System.

## Features
- **Professional Umpire Panel**: Real-time scoring and clock management.
- **OBS Overlay**: Lower-third scoreboard with integrated raid timers.
- **LED Display**: High-visibility audience visuals.
- **Firebase Sync**: Instant synchronization across all devices.

## Run Locally
1. Install dependencies: `npm install`
2. Start the app: `npm run dev`
